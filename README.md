# Rust Dependency Analyzer with Visualization

Анализатор и визуализатор зависимостей Rust пакетов.

## Описание

Скрипт получает информацию о зависимостях Rust пакетов из crates.io и создает графическое представление графа зависимостей с использованием Graphviz.

## Особенности этапа 5

- **Graphviz визуализация**: Генерация DOT представления графа зависимостей
- **Автоматическое создание изображений**: PNG файлы графов зависимостей
- **Сравнение с Cargo**: Анализ расхождений с штатными инструментами
- **Три примера**: Демонстрация для пакетов serde, tokio, reqwest
