# Ассемблер и Интерпретатор УВМ - Этап 2

## Новые возможности этапа 2:

### 1. Раздельная память
- **Память команд**: хранит исполняемый код
- **Память данных**: хранит данные программы

### 2. Интерпретатор с дампом памяти
- Выполнение программ из промежуточного представления
- Сохранение дампа памяти в формате JSON
- Поддержка диапазонов адресов для дампа

### 3. Тест копирования массива
- Автоматический тест работы с памятью
- Демонстрация корректности операций чтения/записи

## Использование

### Ассемблирование в промежуточное представление

python cli.py assemble-int test_program.json program.intermediate.json
