# Rust Dependency Analyzer

Анализатор зависимостей Rust пакетов для этапа 2 задания.

## Описание

Скрипт получает информацию о прямых зависимостях Rust пакетов из репозитория crates.io для заданных версий.

## Особенности

- Использует официальное API crates.io
- Не использует менеджеры пакетов или сторонние библиотеки для анализа зависимостей
- Получает информацию для конкретных версий пакетов
- Выводит все прямые зависимости в консоль
